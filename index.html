Alguns Comandos de Github

1. Git init

É utilizado para criar um novo repositório Git em um diretório existente. Isso marca o início do rastreamento de alterações no projeto.

2. Git clone

Este comando movimenta os arquivos do espaço remoto no servidor para o seu espaço local em seu computador.

3. Git add

Este comando é utilizado em seu espaço de trabalho local em seu computador.

4. Git commit

Este comando movimenta o arquivo de seu espaço LOCAL para o epaço de STAGE em seu computador.

5. Git pull

Este comando movimenta as mudanças do espaço REMOTO no servidor para seu espaço LOCAL em sua máquina.
